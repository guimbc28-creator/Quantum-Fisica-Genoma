# Unified Information Core

Sistema experimental de simulação de informação unificada entre três domínios:

- Computação Quântica (simulada)
- Física da Informação (entropia e energia)
- Genoma (mutação e complexidade)

---

## ⚙️ Como funciona

O sistema recebe:

- Task → define o domínio
- Data → objeto analisado

Pipeline:

Router → InfoField → Transformer → Domain Module → Output

---

## 🚀 Como executar

```bash
python main.py
